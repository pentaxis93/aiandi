# AI and I

A blog about human-AI collaboration at [aiandi.dev](https://aiandi.dev).

## What is this?

This is the source for aiandi.dev - a blog exploring what happens when a 56-year-old learns to code through AI collaboration. Not tutorials. Not hot takes. Stories about what actually works, what doesn't, and what it feels like to build real things with an AI partner.

## Tech Stack

- [Astro](https://astro.build) - Static site generator
- [Tailwind CSS v4](https://tailwindcss.com) - Styling
- [MDX](https://mdxjs.com) - Content
- Deployed to [aiandi.dev](https://aiandi.dev)

## Development

```bash
bun install
bun run dev
```

Articles progress through a structured pipeline from idea to publication. See `PIPELINE.md` for the full workflow.
